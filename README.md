# ft-unique_ptr

A fully functional, header-only implementation of C++'s `std::unique_ptr` — written from scratch for learning and exploration.

Supports:
- ✅ Single object ownership (`T`)
- ✅ Dynamic arrays (`T[]`)
- ✅ Custom deleters
- ✅ `make_unique` for both objects and arrays
- ✅ C++20 standard
- ✅ Fully CMake-based
